# PCCXAI PM Assistant

GitHub-native PM assistant for the PCCXAI ecosystem.

This account is used as a lightweight machine account for project operations around the PCCXAI repositories.

It helps with repository status tracking, issue and pull request triage, release readiness checks, documentation review, and contributor onboarding support.

Final maintainer decisions remain with [hkimw](https://github.com/hkimw).

## Scope

This assistant account may help with:

- repository status tracking
- issue and pull request triage
- documentation review
- release readiness review
- contributor onboarding support
- project board maintenance support

## Boundaries

This account does not make final project decisions.

The following actions require maintainer approval:

- merging pull requests
- publishing releases
- creating release tags
- changing repository rulesets
- changing required checks
- changing security policy
- making major roadmap decisions

## Repository focus

Primary PCCXAI repositories:

- [pccx](https://github.com/pccxai/pccx)
- [pccx-lab](https://github.com/pccxai/pccx-lab)
- [pccx-FPGA-NPU-LLM-kv260](https://github.com/pccxai/pccx-FPGA-NPU-LLM-kv260)
- [pccxai organization profile](https://github.com/pccxai/.github)

## Operating principle

GitHub remains the source of truth.

Issues, pull requests, discussions, project boards, releases, and CI results are the basis for project decisions.

This account reduces project management overhead without replacing maintainer review.
